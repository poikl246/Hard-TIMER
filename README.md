# Hard-TIMER
Это шуточный таймер созданный ради шутки. 


# Start timer
```bash
git clone https://github.com/poikl246/Hard-TIMER
cd Hard-TIMER
docker-compose up 
```

# Stop timer

CTRL + C 

или 

```bash
docker-compose down
```
Выполенный в папке с проектом. 



# Reset timer 
Для сброса таймера нужно удалить docker volume
